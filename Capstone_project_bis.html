Cap Stone project

Goal is to apply the learned skills to an application project covering the following steps:

  • understanding of the volume and complexity of the datas
  • cleaning principles for the train set file
  • building of the model “Katz Back off”, with in the usage of n-grams, then we will use a Shiny app to demo the tool

The datas are large and are composed of 3 sets related to texts on Blogs, News and Twitter

The following main packages from R were used (among others):
- data.table
- quanteda for text mining. n-grams are stored as .rds files
- sqldf to access quickly to the search results

An overview of the corpus composed

  • The files are selected using a ratio 1/100 (1%) and cleaned off their symbols, capital letters, ponctuations, “stopwords”…
  • Individual tokens and n-grams (n=2,3,4) are generated and stored as .rds files

Strategy for definition of the next word

The strategy being used is the Katz’s back-off model or n-grams.
Here is a 3-grams example extracted from a line:

## [1] "vowing to never"    "to never freeze"    "never freeze again"

A simple strategy is for example filtering the n-grams with the input of the n-1 words to easily define the outcome with the highest probability for the next word

According to the Naives Bayes’s theorem:

P( B | A ) = [ P( B) X P( A | B) ] / P(A)

or said differently

P(this clean water) = P(water | this clean) X P(this clean)

New word found

Assuming we find a high probability of the fourth word knowing the first 3 words, we display the up-coming fourth word

And continuing, if not found, we use the 3-grams

Until the 2-gram.

If not found, a random selection of the most used word is made (to avoid the same proposal all the times)

A step further with LDA (Quanteda)

LDA stands for Latent Dirichlet Allocation
It belongs to the topic modeling side (bundle of correlated terms)

Convert the trimmed DFM to a topic_models object
dfm2topicmodels < convert(mydfm.un.trim, to = “topicmodels”)
And the model:
lda.model < LDA(dfm2topicmodels, topic.count)
